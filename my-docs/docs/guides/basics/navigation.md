# Navigation in Docusaurus

Learn how to set up and customize navigation in your Docusaurus site.

## Navbar

The navbar appears at the top of every page and provides main navigation.

### Basic Navbar Configuration

In `docusaurus.config.js`, configure the navbar:

```javascript
themeConfig: {
  navbar: {
    title: 'My Site',
    logo: {
      alt: 'My Site Logo',
      src: 'img/logo.svg',
    },
    items: [
      {
        type: 'docSidebar',
        sidebarId: 'tutorialSidebar',
        position: 'left',
        label: 'Tutorial',
      },
      { to: '/blog', label: 'Blog', position: 'left' },
      {
        href: 'https://github.com/my-org/my-repo',
        label: 'GitHub',
        position: 'right',
      },
    ],
  },
}
```

### Navbar Items

#### Doc Sidebar
Links to documentation with sidebar:

```javascript
{
  type: 'docSidebar',
  sidebarId: 'tutorialSidebar',
  position: 'left',
  label: 'Tutorial',
}
```

#### Internal Links
Links to internal pages:

```javascript
{ to: '/blog', label: 'Blog', position: 'left' }
```

#### External Links
Links to external sites:

```javascript
{
  href: 'https://github.com/my-org/my-repo',
  label: 'GitHub',
  position: 'right',
}
```

#### Dropdowns
Create dropdown menus:

```javascript
{
  type: 'dropdown',
  label: 'Community',
  position: 'left',
  items: [
    {
      label: 'Discord',
      href: 'https://discord.gg/my-server',
    },
    {
      label: 'Twitter',
      href: 'https://twitter.com/my-account',
    },
  ],
}
```

## Sidebar

The sidebar provides navigation within documentation sections.

### Sidebar Configuration

Create `sidebars.js` in your project root:

```javascript
module.exports = {
  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Getting Started',
      items: [
        'getting-started/installation',
        'getting-started/configuration',
        'getting-started/first-project',
      ],
    },
    {
      type: 'category',
      label: 'Guides',
      items: [
        'guides/basics/navigation',
        'guides/basics/sidebar',
      ],
    },
  ],
};
```

### Sidebar Items

#### Documents
Link to individual docs:

```javascript
'intro'
```

#### Categories
Group related docs:

```javascript
{
  type: 'category',
  label: 'Getting Started',
  items: [
    'getting-started/installation',
    'getting-started/configuration',
  ],
}
```

#### Links
External links in sidebar:

```javascript
{
  type: 'link',
  label: 'GitHub',
  href: 'https://github.com/my-org/my-repo',
}
```

#### Auto-generated
Auto-generate sidebar from file structure:

```javascript
tutorialSidebar: [
  {
    type: 'autogenerated',
    dirName: '.',
  },
]
```

## Footer

The footer appears at the bottom of every page.

### Footer Configuration

```javascript
themeConfig: {
  footer: {
    style: 'dark',
    links: [
      {
        title: 'Docs',
        items: [
          {
            label: 'Tutorial',
            to: '/docs/intro',
          },
          {
            label: 'API',
            to: '/docs/api/cli',
          },
        ],
      },
      {
        title: 'Community',
        items: [
          {
            label: 'Discord',
            href: 'https://discord.gg/my-server',
          },
          {
            label: 'Twitter',
            href: 'https://twitter.com/my-account',
          },
        ],
      },
    ],
    copyright: `Copyright © ${new Date().getFullYear()} My Project, Inc.`,
  },
}
```

## Breadcrumbs

Docusaurus automatically generates breadcrumbs for documentation pages.

To customize breadcrumbs, modify the theme components.

## Search Navigation

The search bar allows users to quickly find content.

Configure search in the navbar or use plugins like Algolia.

## Mobile Navigation

Docusaurus automatically provides a mobile-friendly navigation menu.

The hamburger menu collapses the navbar on small screens.

## Custom Navigation

For advanced customization, you can swizzle navigation components:

```bash
npm run swizzle @docusaurus/theme-classic Navbar
```

Then modify `src/theme/Navbar.js`.

## Best Practices

- Keep navigation simple and intuitive
- Use clear, descriptive labels
- Group related items in categories
- Limit navbar items to prevent clutter
- Test navigation on mobile devices
- Use consistent styling across all navigation elements
